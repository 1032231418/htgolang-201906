2. 猜数字游戏
    for if continue/break
    a. 生成一个[0, 100)随机数
    b. 让用户最多猜 5次（从命令行输入一个整数）
        猜的太大 => 提示太大了，你还有N此猜测机会
        猜的太小 => 提示太小了，你还有N此猜测机会
        猜中了 => 猜中了

        5次都没猜中 => 退出，并提示太笨了


    挑战：
        当5此都没有猜中：提示太笨了，重新开始猜数游戏（重新生成随机数，重新开始5次计数）
