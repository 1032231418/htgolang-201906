其他详情：

- 1, 对输入年月日电话范围限制
    - 输入必须为int类型
	- 年：满足4位，小于2019
	- 月：小于12
	- 日 ：小于31
	- tel： 满足7位数或者11位数
- 1.1 用户名重复判断
	- 用户名重复必须重新输入
- 2, 调用输入的日期计算年龄
- 3，简单实现了部分timeout功能，60秒钟不输入自动断开
